Capture

Hooks record session activity while a background daemon parses transcripts and extracts spores — decisions, gotchas, discoveries, and trade-offs — automatically.

Connect

A living knowledge graph links sessions to spores, spores to entities, and patterns to wisdom. Agents traverse it via MCP tools; humans explore it in the dashboard.

Search

Vector embeddings and full-text search work together. Relevant spores surface automatically with each prompt — no manual searching required.
